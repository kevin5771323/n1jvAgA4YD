# 小型企业办公自动化系统 java869

## 项目概述

本项目是一款面向小型企业的办公管理系统，旨在通过信息化手段，提高企业内部管理的效率与保密性，降低管理成本。

## 技术栈

- **前端：** Vue.js
- **后端：** Java语言，使用SSM（Spring、SpringMVC、MyBatis）框架
- **数据库：** Mysql

## 功能模块

### 系统角色

- 管理员
- 部门经理
- 总经理
- 员工

#### 管理员

- 管理部门经理、总经理、员工信息
- 管理员工留言与工资
- 管理内部邮件与审批流程
- 管理离职申请

#### 部门经理

- 发放员工工资
- 审核并回复员工留言
- 管理员工工资
- 审核离职申请信息
- 查询和下载内部邮件及审批流程

#### 总经理

- 查询和下载内部邮件及审批流程
- 审核员工离职申请
- 查询员工工资与信息
- 查询员工与部门经理

#### 员工

- 发布留言与内部邮件
- 提交离职申请
- 查询通知公告和审批流程
- 查看个人工资

## 运行环境

- **开发环境：** Eclipse或IntelliJ IDEA
- **操作系统：** 不限
- **依赖环境：** Java环境，Mysql数据库

## 部署步骤

1. 配置Java开发环境与Mysql数据库
2. 导入项目至开发工具（Eclipse或IntelliJ IDEA）
3. 创建数据库，执行项目提供的SQL脚本以初始化数据表
4. 编译项目
5. 部署至服务器
